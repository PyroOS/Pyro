Introduction

Address is a plugin for Syllable's Dock. This plugin will open Webster to a specified Address. Address can also search through Yahoo, Google, and many other websites. With Address you can also specify websites to search from. You can also specify a website that and give it a key word or "tag" to quickly go to a favorite website. Address also contains a buffer so that all searches and websites you go to will be saved.


How do I use?

Using Address is pretty simple. I assume you know how to install and load Dock plugins. From here you will see a DropdownMenu Box on your Dock. Type "help" in the TextBox and hit "Enter" on your keyboard. This will open this file in Webster. To use a search, type the search tag('g:' for google, 'y:' for yahoo, and so on) followed by the item/items you want to search. To open a website other than a search, type one of the following: 'http://website.com', or 'http://www.website.com', or 'www.website.com' You cannot just type the website(ie: website.com), because that will use the default search to search for that website. Thats it folks :)


What are the default searches?

The default searches are(remember the first part is the tag used to search the website):
g: which will search through http://www.google.com with the specified search.
w: which will search through http://www.wikipedia.org with the specified search.
y: which will search through http://www.yahoo.com with the specified search.
th: which will search through http://thesaurus.reference.com with the specified search.
dic: which will search through http://dictionary.reference.com with the specified search.


How do I add my own searches?

Adding searches/websites is really easy. Type 'settings' in the DropdownMenu Box and hit enter. This will open the 'Settings Dialog'. From here, all you have to do is click the 'New' button. This will popup a 'New' window.


What other settings are there?

Address comes with a built in buffer so all websites/searches can be saved. The buffer is saved when the plugin is closed or when Dock closes. You can clear this buffer at any time by hitting "clear" on the keypad and then hitting enter. The other feature of Address is that has a default search engine which you can change at any time. As of right now you can only change between the default search engines, but that will change in due time :) The only other setting that Address has is an about box. Please don't be shy to take a look at it :)
